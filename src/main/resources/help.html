О системе моделирования движения транспорта

Данная система предназначена для моделирования движения транспортных средств на различных типах автодорог, таких как городская дорога, автомагистраль или тоннель.

Основные возможности:

  • Настройка параметров автодороги: тип, длина, количество полос и направлений.
  • Для тоннелей: настройка длительности фаз светофоров.
  • Настройка параметров транспортного потока:
    • Детерминированный поток: задание интервала появления и скорости автомобилей.
    • Случайный поток: выбор закона распределения (равномерный, нормальный, показательный) и его параметров для времени появления и скорости автомобилей.
  • Интерактивное добавление до двух светофоров на дорогу с настройкой фаз и направления действия.
  • Интерактивное добавление дорожных знаков (4-х типов) с указанием направления действия.
  • Визуализация движения транспортных средств в реальном времени с отображением их скорости.
  • Отображение оставшегося времени работы светофоров.
  • Управление скоростью симуляции (1x, 2x, 3x).
  • Запуск, пауза и остановка процесса моделирования.

Технологии:

Приложение разработано на языке программирования Java с использованием библиотеки Swing для графического интерфейса пользователя.
